VM搭建CentOS7-并安装docker

虚拟机 linux
发布日期:   2024-07-28
更新日期:   2024-07-28
阅读时长:   5 分
阅读次数:  

本文涉及到VMware安装linux，配置网络，安装docker。适用于开发JAVA项目等。

1 VMware安装Linux

  1. 左上角【文件】，点击新建虚拟机，选择自定义，典型的较为简单，但是我们选择自定义安装

  2. 点击【下一步】选择【稍后安装操作系统】

  3. 然后我们选择安装Linux系统，并选择相应的版本，示例使用的是CentOS 7 64位

  4. 随意命名并选择相应的位置。

  5. 处理器，内存看个人需求，我设置1个处理器，每个处理器2内核，2GB内存。

  6. 网络类型，选择NAT的方式

  7. 选择推荐一直【下一步】即可。

  8. 磁盘，我们选择创建新的磁盘，然后下一步

  9. 指定磁盘容量的大小，这里默认20GB。不用担心大小，只要不勾选【立即分配所有磁盘空间】即可，它是用多少申请多少的磁盘空间。

  10. 然后【下一步】—->并【完成】安装配置。安装完成后，可以先配置VMware的网络，详细看第二部分。

  11. 接下来，安装系统。双击【CD/DVD(IDE)】然后选择【使用ISO映像文件】，这里与上面所选择的版本对应，我这里使用的是CentOS-7-x86_64-DVD-1804.iso。

  12. 开启虚拟机等待系统安装程序，选择中文点击继续。

  13. 时区语言自己选。我这里是亚洲中文，点击软件选择。选择如图所示的两项，如果想要可视化界面可以选择GNOME桌面。

  14. 点击安装位置，点完成即可。

  15. 点击网络和主机名，设置主机名，点击右上角打开以太网，当前是dhcp自动分配，一会儿安装完毕进入系统配置。

  16. 点击开始安装后，点ROOT密码，设置root账户的密码，点击完成。想要创建别的用户也可以点击创建用户进行创建。等待安装完毕点击重启。

2 网络配置

2.1 虚拟机网络配置

  1. 上方菜单【编辑】-【虚拟网络编辑器】

  2. 点击VMnet8。选择NAT模式，配置子网为，192.168.x.0，子网掩码255.255.255.0。x最好不要设置为1。点击确定

2.2 本机网络配置

  1. 进入windows的设置，找到网络和Internet，点击更改适配器选项。

  2. 右键VMnet8-【属性】，点击【Internet协议版本4】，设置192.168.x与上方2.1设置的192.168.x相同。第四位如图所示。

2.3 CentOs网络配置

  1. 输入root账号密码，密码输入不会回显，正常输入按回车即可。ip addr获得网卡信息。

  2. 切换到网卡的目录，vim编辑网卡。

    ifcfg-[网卡名]

cd /etc
cd /sysconfig
cd /network-scripts
vim ifcfg-ens33
或
vim /etc/sysconfig/network-scripts/ifcfg-ens33

  3. 按i键进入编辑模式，将配置修改为如下配置，IPADDR为你想要的本机的IP地址，主要观察

    BOOTPROTO设置为static

    ONBOOT设置为yes

    配置后5行。

    按下【esc】后，按【:】,输入:wq保存

    TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="static"
DEFROUTE="yes"
IPV4_FAILURE_FATAL="no"
IPV6INIT="yes"
IPV6_AUTOCONF="yes"
IPV6_DEFROUTE="yes"
IPV6_FAILURE_FATAL="no"
IPV6_ADDR_GEN_MODE="stable-privacy"
NAME="ens33"
UUID="d10f88c0-6334-4854-b50a-1b2432d39f5a"
DEVICE="ens33"
ONBOOT="yes"
IPADDR="192.168.5.130"
PREFIX="24"
GATEWAY="192.168.5.2"
DNS1="8.8.8.8"
IPV6_PRIVACY="no"

  4. 输入reboot重启，尝试使用ping命令ping www.baidu.com，能ping通即为成功。

3 安装docker

  1. 卸载旧版本

    yum remove docker \
            docker-client \
            docker-client-latest \
            docker-common \
            docker-latest \
            docker-latest-logrotate \
            docker-logrotate \
            docker-engine

  2. 安装yum工具包

    yum install -y yum-utils device-mapper-persistent-data lvm2

    这一步骤可能会出现如下问题：

    Could not retrieve mirrorlist http://mirrorlist.centos.org/?release=7&arch=x86_64&repo=os&infra=stock error was
14: curl#6 - "Could not resolve host: mirrorlist.centos.org; 未知的错误"

    解决办法如下：[yum:”Could not retrieve mirrorlist”故障解决](https://kylece666.github.io/2024/07/28/yum:"Could not retrieve mirrorlist”故障解决/)

  3. 设置阿里云docker-ce镜像源

    yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

  4. 安装docker

    yum install docker-ce docker-ce-cli containerd.io

  5. 启动docker并设置开机自启动

    systemctl start docker

systemctl enable docker

  6. 解决目前docker hub国内无法访问

    sudo mkdir -p /etc/docker

sudo tee /etc/docker/daemon.json <<-'EOF'
{
    "registry-mirrors": [
        "https://do.nark.eu.org",
        "https://dc.j8.work",
        "https://docker.m.daocloud.io",
        "https://dockerproxy.com",
        "https://docker.mirrors.ustc.edu.cn",
        "https://docker.nju.edu.cn"
    ]
}
EOF

sudo systemctl daemon-reload
sudo systemctl restart docker
